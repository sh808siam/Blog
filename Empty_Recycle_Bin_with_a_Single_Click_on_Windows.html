Empty Recycle Bin with a Single Click on Windows

<< blog

PowerShell Command is :

powershell.exe -command "& {Clear-RecycleBin -Force}"

[ or ]

powershell.exe -command Clear-RecycleBin -Force

Tip : You can remove "-Force" from the shell command to get Delete Confirmation Dialogue Box.

Now that you know the command, let's make the shortcut. Firstly, Go to context menu / option and click on "new". Afterwards, select shortcut and paste any of codes given at the top. Click "next" and give a name for that shortcut as you like. For example, EmptyRBin

Now you can double click on that shortcut to clear or empty your recycle bin. Now, let's go to the final step. Select the shortcut then open context menu / option. After that select the option "pin to taskbar."

DONE! Now you can clear recycle bin without going through multiple mouse clicks just by clicking on the taskbar icon.
