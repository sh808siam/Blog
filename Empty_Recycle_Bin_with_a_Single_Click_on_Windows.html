Empty Recycle Bin with a Single Click on Windows

<< blog

PowerShell Command is :

powershell.exe -command "& {Clear-RecycleBin -Force}"

[ or ]

powershell.exe -command Clear-RecycleBin -Force

Tip : You can remove "-Force" from the shell command to get Delete Confirmation Dialogue Box.

Now that you know the command, let's make a shortcut. Firstly, Go to context menu / option and click on "new". Afterwards, select shortcut and paste any of codes given at the top. Click "next" and give a name for that shortcut as you like. For example, EmptyRBin

You can double click on that shortcut to clear or empty your recycle bin but let's take a step further. Select the shortcut and open context menu / option. After that select "pin to taskbar."

DONE! Now you can clear recycle bin without going through multiple mouse clicks, just single click on the taskbar icon.
